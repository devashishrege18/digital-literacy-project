## Project Overview

This project was developed as part of the Digital Literacy course assignment to demonstrate practical understanding of essential digital skills required for students in academic and professional environments. The project covers five modules including digital awareness, professional digital identity, coding and collaboration platforms, email etiquette, and cybercrime awareness.

The repository contains all task-wise deliverables, screenshots, written reflections, and supporting files prepared according to the assignment brief.
